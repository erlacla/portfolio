Write a JavaScript function that reverse a number.
Example x = 32243; Expected Output : 34223

  • 3 types of functions are function expressions, function declarations, and arrow functions. Function expressions have a const or let to assign the function to a value. They also need a semicolon at the end. They can assign an anonymous function to a variable or a named function to a variable. Function declarations declare a variable to be a function and they are hoisted. Arrow functions do the same as a declaration just with less typing. They are always anonymous and best used for short functions that don't have many statements in the body.
  • A function can have many parameters or none at all.
  • There are 6 primitive data types: strings, numbers, booleans, symbols, null, and undefined.
  • Non-primitive data types are: arrays, functions, and objects.
  • If you don't use let or const, a variable will be global.
  • Type coercion = when JavaScript tries to change a string to a number automatically.
  • Verbose = unneeded, too wordy. DRY = don't repeat yoursef.
  • Ternary operator is: ? can be used in place of an if else block.
    condition ? (code for true) : (code for false)
  • For - of loop can be used for arrays.
    for(const value of array) { console.log(value); }
  • Rest operator = function rest(...args) { return args; }
